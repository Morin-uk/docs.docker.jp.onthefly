NIST ITL October 2017 guidance on application container security

The NIST Information Technology Laboratory (ITL) Bulletin dated October 2017, NIST Guidance on Application Container Security, is a summarization of the content made available in NIST SP 800-190 and NISTIR 8176. It provides a fundamental high-level starting point from which to evaluate the primary security enhancements that containers provide. It is not specific to any one container runtime.
