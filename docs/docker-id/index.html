Docker ID アカウント

フリーの Docker ID を取得すると Docker Hub リポジトリへアクセスでき、ベータプログラムのいくつかを利用できます。 Docker ID の取得には電子メールアドレスのみが必要です。

Docker ID の登録

Docker ID は、提供されている Docker サービスにおける各ユーザー向けの名前となり、Docker フォーラムでのユーザー名にもなります。 新たな Docker ID は以下のようにして生成します。

  1. Docker Hub サインアップページにアクセスします。

  2. ユーザー名を入力します。これが Docker ID となります。

    Docker ID は 4 文字以上、30 文字までで、数字と英小文字のみを用います。

  3. ユニークで適正なメールアドレスを入力します。

  4. パスワードを入力します。 なおパスワードは最低 9 文字以上なければなりません。

  5. Captcha 確認を行って Sign up をクリックします。

    Docker からの確認メールが、入力されたメールアドレスに送信されます。

  6. メールアドレスを確認して登録操作を完了します。

メモ: メールアドレスの承認処理を終えてからでないと Docker ID を使ったログインはできません。

ログイン

Docker ID の登録を行ってメールアドレスの承認を終えたら Docker Hub にログインできるようになります。

docker login コマンドを使って CLI を通じたログインもできます。 詳しくは こちら を参照してください。）

警告: docker login コマンドを使ってログインすると、認証情報がホームディレクトリ配下の.docker/config.jsonに保存されます。 そしてパスワードは base64 エンコードにより保存されます。

より強力なセキュリティが必要である場合は パーソナルアクセストークン（personal access token）を使ってログインすることもできます。 この場合もパスワードはエンコードされてファイルに保存されますが、管理操作（たとえばパスワード変更など）を行うことはできません。 パスワードやパーソナルアクセストークンを安全に保存する必要がある場合は Docker credential helpers を利用してください。
