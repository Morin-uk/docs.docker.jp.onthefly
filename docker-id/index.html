Docker ID アカウント

読む時間の目安: 1 分

フリーの Docker ID を取得すると Docker Hub リポジトリへアクセスでき、ベータプログラムのいくつかを利用できます。 Docker ID の取得には電子メールアドレスのみが必要です。

このアカウントがあると、たとえば Docker Support Center、Docker フォーラム、Docker Success ポータルのようなサービスにもログインできるようになります。

Docker ID の登録

Docker ID は、提供されている Docker サービスにおける各ユーザー向けの名前となり、Docker フォーラムでのユーザー名にもなります。

  1. Docker Hub サインアップページにアクセスします。

  2. ユーザー名を入力します。これが Docker ID となります。

    Docker ID は 4 文字以上、30 文字までで、数字と英小文字のみを用います。

  3. ユニークで適正なメールアドレスを入力します。

  4. パスワードを 6 文字以上 128 文字以下の範囲で入力します。

  5. Sign up をクリックします。

    Docker からの確認メールが、入力されたメールアドレスに送信されます。

  6. 受信したメール内のリンクをクリックしてメールアドレスを承認します。

メモ: メールアドレスの承認処理を終えてからでないと Docker ID を使ったログインはできません。

ログイン

Docker ID の登録を行ってメールアドレスの承認を終えたら Docker Hub や Docker Support にログインできるようになります。

docker login コマンドを使ってログインすることもできます。 （docker loginの詳細はこちらを参照してください。）

警告: docker login コマンドを使ってログインすると、認証情報がホームディレクトリ配下の.docker/config.jsonに保存されます。 そしてパスワードは base64 エンコードにより保存されます。 より安全なパスワード保存を必要とする場合は Docker credential helpers を利用してください。
